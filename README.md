# 📝 Note Taking App

A modern desktop note-taking application built with C# and .NET, designed to help users create, organize, edit, and manage their notes efficiently. The application features a clean and intuitive user interface powered by Bunifu UI and DotNetBar components, with data persistence handled through Microsoft Access.

## ✨ Features

- Create and save notes
- Edit existing notes
- Delete notes
- Organized note management
- Fast and responsive desktop interface
- User-friendly design
- Persistent data storage using Microsoft Access

## 🛠️ Built With

- C#
- .NET Framework
- Microsoft Access Database
- Bunifu UI Framework
- DotNetBar Components
- Windows Forms

## 🚀 Getting Started

### Prerequisites

- Visual Studio 2022 or newer
- .NET Framework
- Microsoft Access Database Engine

### Installation

1. Clone the repository:

```bash
git clone https://github.com/AliSaadatDev/NoteTakingApp.git
```

2. Open the solution file in Visual Studio.

3. Restore project dependencies if needed.

4. Build and run the application.
